# How to Contribute 如何贡献

There are just a few small guidelines you need to follow.
遵守以下准则。

## Code guidelines 代码规范

Do not use deprecated APIs for the Go version specified in go.mod.
不能使用 go.mod 中对应go版本过时的API。

## Document 文档规范

Update docs/change.md after changes.
功能变动之后需要更新文件到 docs/change.md 中。

## Code review 代码审阅

After completing each feature, ensure that unit tests pass.
每一个功能完成后，要确保单元测试通过。
